# Buona Pizza - Landing Page Estática

Este projeto foi convertido em uma **landing page estática de alta performance**, utilizando apenas HTML, Vanilla CSS e JavaScript puro.

## Estrutura do Projeto
- `index.html`: Estrutura principal da página.
- `style.css`: Estilização completa com design system premium.
- `assets/`: Pasta contendo todas as imagens e vídeos utilizados.

## Como Visualizar
Basta abrir o arquivo `index.html` em qualquer navegador ou utilizar um servidor estático simples.

### Exemplo com Python:
```bash
python3 -m http.server 8000
```
Acesse `http://localhost:8000` no seu navegador.

## Destaques do Design
- **Estética Heritage**: Combinação de cores clássicas italianas (verde profundo, creme e vermelho tomate).
- **Tipografia**: Uso de fontes Serif (Playfair Display) para um toque artesanal e luxuoso.
- **Micro-animações**: Transições suaves e efeitos de scroll para uma experiência premium.
- **Responsividade**: Totalmente otimizado para dispositivos móveis.
